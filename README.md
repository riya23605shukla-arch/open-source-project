# Open Source Project
Open Source Scripts Project

This project is a collection of basic Bash scripts created as part of an open-source learning assignment. The goal of this project is to understand Linux commands, scripting, and version control using Git and GitHub.

# Project Overview

In this project, I created and executed multiple Bash scripts on a Linux (WSL Ubuntu) environment. Each script performs a specific task related to system operations and automation.

# Description of Scripts
🔹 Script 1

Performs basic operations using Linux commands. Helps in understanding command execution and scripting basics.

🔹 Script 2

Focuses on system-level tasks and improves familiarity with shell scripting.

🔹 Script 3 – Directory Audit
Displays permissions and size of important directories
Checks Git configuration status
🔹 Script 4 – Log File Analysis
Searches for keywords in system logs
Uses dmesg when log files are not available
Displays matching lines
🔹 Script 5 – Open Source Manifesto Generator
Takes user input
Generates a personalized manifesto
Saves output in a .txt file
